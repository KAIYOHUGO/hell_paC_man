# Code Design

`V{Structure Name}` is a Trait (VTable)
`P{Structure Name}` is a Pointer of VTable (`self*, VTable*`)
`C{Structure Name}` is a Collection of method of `{Structure Name}`
`hook_{Function Name}` is a Hook call by engine
